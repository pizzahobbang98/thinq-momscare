# ThinQ Mom 발표자료용 문장

PPT·포스터·산출물에 **그대로 붙여 넣을 수 있는** 짧은 문장 모음입니다.  
상세 설명은 [service-overview.md](./service-overview.md)를 참고하세요.

---

## 1. 한 줄 소개

- ThinQ Mom은 임산부의 **말 한마디**를 집 안 환경과 가족의 행동으로 바꾸는 **ThinQ ON 기반 케어 서비스**입니다.
- **말 한마디로** 임산부의 하루를 돌보는 ThinQ ON 기반 케어 경험

---

## 2. 문제 정의

- 임산부의 어려움은 **냄새, 피로, 수면, 답답함**처럼 일상 환경과 연결되어 **반복**되지만, 기존 스마트홈은 이를 **생활 맥락**으로 해석하지 못합니다.
- 매번 가전을 조작하거나 가족에게 설명해야 하는 **부담**이 있습니다.
- 남편은 도와주고 싶어도 **지금 무엇을 해야 할지** 모릅니다.

---

## 3. 핵심 인사이트

- 임산부는 **“입덧모드 켜줘”**보다 **“냄새 때문에 못 먹겠어”**라고 말합니다.
- 중요한 것은 기기 제어가 아니라 **힘든 순간의 조건**을 바꾸는 것입니다.
- 남편에게 필요한 것은 상태 정보 전체가 아니라 **오늘 할 수 있는 행동 제안**입니다.
- **허브**는 말을 모드로, **앱**은 결과를 케어와 기록으로 남깁니다.

**필수 포함 문장**

> **기존 ThinQ가 가전을 움직였다면, ThinQ Mom은 임산부가 힘든 순간의 조건을 바꿉니다.**

---

## 4. 서비스 컨셉

- ThinQ Mom은 **자연어 발화**를 AI가 해석해 **입덧·수면·휴양지** 케어 모드로 전환하고, 아내와 남편에게 **각각 필요한 정보**를 전달합니다.
- **공기청정기 제어**가 아니라, **생활 환경과 가족 행동을 함께 바꾸는** 임산부 케어 서비스입니다.

---

## 5. 핵심 기능 (5개)

1. **AI 발화 해석** — 문맥·감정 신호 기반 케어 모드 분류 (입덧 / 수면 / 휴양지)
2. **ThinQ ON 허브** — 미니멀 첫 화면, 음성·텍스트·예시 칩 실행
3. **모드별 환경 전환** — 공기청정기 실제 제어 + 3D scene 연동
4. **엄마품 화면** — 오늘의 케어, 초음파 갤러리, AI 자동 다이어리
5. **아빠손길 화면** — 증상 비노출, **행동 중심** 추천·집안일 제안

---

## 6. 시연 플로우

1. **Hub** — ThinQ ON 이미지만 → 터치 → 패널 오픈  
2. **입덧** — “나 지금 입덧이 너무 심해” → 공기청정기 → 3D → wife/husband  
3. **수면** — “나 이제 잘 준비 할래” → 수면 환경 → SLEEP_SCENE  
4. **휴양지** — “바다 보면서 쉬고 싶어” → RESORT_SCENE  
5. **엄마품** — 오늘의 케어 · 초음파 · 다이어리  
6. **아빠손길** — 오늘의 추천 · 집안일 (행동 제안)  
7. **부수** — 초음파 업로드 · 과일 비유 · 다이어리 생성  

---

## 7. 차별점

| 기존 ThinQ | ThinQ Mom |
|-----------|-----------|
| 기기 선택 · 명령 | **생활 언어** 한마디 |
| 명령어 해석 | **문맥·감정** 해석 |
| 단일 가전 제어 | 환경 + **가족 행동** + 기록 |
| 기기 로그 | 케어 카드 · 초음파 · 다이어리 |

- **기존 ThinQ가 가전을 움직였다면, ThinQ Mom은 임산부가 힘든 순간의 조건을 바꿉니다.**

---

## 8. 기대효과

**임산부**

- 설명·조작 부담 감소, 케어·기록 자동화

**가족**

- **무엇을 도울지** 명확, 배려가 **행동**으로 연결

**LG ThinQ**

- 가전 제어 → **생활 케어 플랫폼**, ThinQ ON **허브** 역할 강화

- **가전 제어는 케어가 되고, 가족의 배려는 행동으로 이어집니다.**

---

## 9. 발표 마무리

- ThinQ Mom은 임산부가 **힘든 순간을 설명하지 않아도**, 그 순간의 **조건을 집이 먼저** 바꿔주는 경험입니다.
- ThinQ Mom은 **가전 제어를 넘어**, 임산부와 가족의 하루를 함께 돌보는 **ThinQ ON 기반 케어 경험**입니다.
- **Hub 한 문장 → AI 해석 → 가전 → 3D → 아내·남편 → 기록**까지, 하나의 케어 흐름입니다.
