# 시연 흐름

## 준비 화면

1. 모바일 또는 웹 브라우저에서 Vercel 배포 주소의 `/`를 엽니다.
2. 노트북 브라우저에서 같은 배포 주소의 `/simulation-3d/index.html`을 엽니다.
3. 두 화면 모두 마이크 권한을 허용합니다.
4. 전구 시연이 필요하면 로컬 FastAPI 서버와 ngrok을 먼저 실행합니다.

## 메인 앱 시연

1. `/`에서 사용자 상태를 `임신준비중` 또는 `임신중`으로 선택합니다.
2. 사용자 역할을 `아내` 또는 `남편`으로 선택합니다.
3. 홈에서 현재 상태와 기록 흐름을 확인합니다.
4. 하단 가운데 HUB 버튼을 누른 채 테스트 문구를 말합니다.
5. 음성 인식, 의도 분석, TTS 응답, 공기청정기/전구/3D 상태 변경을 확인합니다.
6. 수동제어 탭에서 공기청정기, 거실조명, 스탠바이미, 냉장고, 로봇청소기, 워시타워 카드 상태를 확인합니다.

## 3D 시뮬레이터 시연

1. `/simulation-3d/index.html`에서 “하이 마더”라고 말합니다.
2. “네 말씀하세요” 응답 이후 케어 명령을 말합니다.
3. 3D 장면, 화면 콘텐츠, 공기청정기 표현, 조명 색상을 확인합니다.
4. TTS/음성 반환 종료 후 10초 동안 추가 발화가 없으면 기본 화면으로 돌아가는지 확인합니다.
5. 기본 화면 복귀 시 메인 앱 수동제어 카드와 기본 대기 조명이 함께 초기화/동기화되는지 확인합니다.

## 권장 시연 순서

1. 임신준비중 아내: “아침 컨디션을 맞춰줘.”
2. 임신준비중 남편: “우리 둘의 저녁을 준비해줘.”
3. 임신중 아내: “음식 냄새 때문에 속이 안 좋아.”
4. 임신중 아내: “잠이 잘 오게 해줘.”
5. 임신중 남편: “빨래와 청소를 도와줘.”
6. 휴양 장면: 바다, 숲, 도시 중 하나 실행
7. 공기청정기 ON/OFF
8. 전구 ON/OFF
9. “기본 모드로 바꿔줘.”

## 시연 전 체크리스트

- Vercel 배포 주소에서 두 URL 모두 접속 가능
- OpenAI API key 설정
- 공기청정기 실제 기기 온라인
- ngrok HTTPS 주소가 Vercel 환경변수와 일치
- FastAPI 서버 8000번 포트 실행
- 마이크 권한 허용
- 임신준비중 5개 모드 동작
- 임신중 6개 모드 동작
- 기본 복귀 후 수동제어 카드와 조명 기본 상태 동기화

현재 기준: Vercel 시연 기능만 반영.
