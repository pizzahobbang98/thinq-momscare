## 허브 입력

- [ ] 허브 버튼을 누르는 동안 진동하는 시각 효과
- [ ] 브라우저 마이크 권한 허용
- [ ] 음성 인식 완료 후 응답이 빠르게 재생
- [ ] 마이크 실패 시 예시 문장 또는 텍스트 입력 가능
- [ ] 오늘의 케어 흐름 2, 3번 문구는 `허브에 말하기`, `3D 기기 반영`으로 유지

## 임신 준비중

- [ ] `아침 컨디션을 맞춰줘.` -> 컨디션 밸런스
- [ ] `수면 리듬을 맞춰줘.` -> 수면 리듬
- [ ] `마음을 환기하고 싶어.` -> 마음 환기
- [ ] `편안하게 쉬고 싶어.` -> 휴식 준비
- [ ] `우리 둘의 저녁을 준비해줘.` -> 둘의 저녁
- [ ] 아내와 남편의 답변·홈 문구가 구분됨

## 임신중

- [ ] `음식 냄새 때문에 속이 울렁거려.` -> `NAUSEA_MODE`, `nausea_food`, 터보
- [ ] `잠이 잘 오게 해줘.` -> `SLEEP_MODE`, `sleep_care`, 수면
- [ ] `빨래와 청소를 도와줘.` -> `HOUSEWORK_MODE`, `housework_care`, 자동
- [ ] `바다를 보며 쉬고 싶어.` -> `TRAVEL_MODE`, `destination_ocean`, 자동
- [ ] 숲과 도시 발화도 각각 다른 3D 장면으로 전환
- [ ] 스탠바이미 화면이 켜져 있어야 하는 장면에서 꺼지거나 깜빡이지 않음
