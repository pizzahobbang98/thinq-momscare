# ThinQ Mom 서비스 개요

## 서비스 정의

ThinQ Mom은 임신을 준비하거나 임신중인 사용자와 배우자의 자연스러운 말을 AI가 이해하고, 공기청정기와 집 안 화면·조명 분위기를 케어 모드로 바꾸는 최종 시연용 MVP입니다.

핵심은 세 화면을 한 번 열어둔 뒤 같은 상태와 실행 결과가 이어지는 모습을 보여주는 것입니다.

| 화면 | URL | 역할 |
| --- | --- | --- |
| 모바일 | `/` | 상태·역할·임신 주차 선택, 홈 요약, 성장 기록, AI 다이어리, 기기 상태 |
| 허브 | `/hub` | 음성·텍스트 입력, 의도 해석, 음성 응답, ThinQ·3D 실행 |
| 3D 홈 | `/simulation-3d/index.html` | 케어 모드에 맞는 화면·조명·공기청정기 장면 표시 |

모바일은 **홈 / 디바이스 2탭만 사용**합니다. 케어 탭과 메뉴 탭은 현재 시연 범위에 없습니다.

## 해결하려는 문제

- 임신 준비중이거나 임신중인 사용자는 컨디션에 따라 집 안 환경을 자주 바꾸고 싶지만 매번 기기를 조작하기 어렵습니다.
- 사용자는 정해진 기기 명령보다 “잠이 잘 오게 해줘”, “냄새 때문에 속이 울렁거려”처럼 생활 언어로 말합니다.
- 배우자는 상대의 상태를 그대로 전달받기보다 지금 할 수 있는 행동을 간단히 안내받는 편이 유용합니다.
- 하루의 대화와 케어 결과를 다시 기록하는 일도 부담이 될 수 있습니다.

## 최종 시연 경험

```text
모바일에서 상태·역할 선택
        ↓
/api/demo-state와 Supabase에 저장
        ↓
허브가 1초마다 상태 반영
        ↓
사용자가 허브에 말함
        ↓
AI 또는 규칙 기반으로 케어 모드 선택
        ↓
음성 응답 + 실제 공기청정기 + 3D 장면 동시 실행
        ↓
모바일 디바이스 탭과 AI 다이어리에 결과 반영
```

## 상태와 역할

- 상태: `임신 준비중`, `임신중`
- 역할: `아내`, `남편`
- 임신중일 때: 1~42주 임신 주차 선택

같은 발화라도 상태와 역할에 따라 답변, 홈 문구, 기록의 관점이 달라집니다.

## 임신 준비중 케어

| 준비 모드 | 발화 예시 | 집 안 변화 |
| --- | --- | --- |
| 컨디션 밸런스 | 아침 컨디션을 맞춰줘. | 자동 공기, 부드러운 아침빛 |
| 수면 리듬 | 수면 리듬을 맞춰줘. | 수면 공기, 화면 자극 감소 |
| 마음 환기 | 마음을 환기하고 싶어. | 자동 공기, 숲 분위기 |
| 휴식 준비 | 편안하게 쉬고 싶어. | 수면 공기, 따뜻한 조명 |
| 둘의 저녁 | 우리 둘의 저녁을 준비해줘. | 자동 공기, 편안한 저녁 장면 |

## 임신중 케어

| 케어 모드 | 발화 예시 | 3D routine | 공기청정기 |
| --- | --- | --- | --- |
| 입덧 케어 | 음식 냄새 때문에 속이 울렁거려. | `nausea_food` | 터보 |
| 수면 케어 | 잠이 잘 오게 해줘. | `sleep_care` | 수면 |
| 가사 케어 | 빨래와 청소를 도와줘. | `housework_care` | 자동 |
| 휴양지 케어 | 바다를 보며 쉬고 싶어. | `destination_ocean` | 자동 |

휴양지는 바다, 숲, 도시 장면을 지원합니다.

## 아침 안내

호출 문구는 **“좋은 아침이야”**입니다.

- 임신 준비중: 조급함을 낮추고 식사, 움직임, 휴식 중 하나를 제안
- 임신중 아내: 모바일에서 선택한 임신 주차를 반영한 행동 제안
- 남편: 배우자의 컨디션을 묻고 도울 행동을 제안

## 실제 ThinQ 연동

허브 케어 실행 시 `/api/thinq/control`과 ThinQ PAT API를 사용해 공기청정기 전원 또는 운전 모드를 변경합니다.

`THINQ_MOCK_FALLBACK=true`이면 실제 기기 연결이 실패해도 3D와 모바일 시연은 mock 결과로 유지됩니다. 실제 기기 성공 여부는 허브의 ThinQ 상태와 `device_events` 저장 여부로 확인합니다.

## 모바일 기능

### 홈 탭

- 상태·역할·임신 주차 선택
- 현재 상태 요약
- 오늘의 케어 흐름
- 임신중 아내의 아기 성장 기록
- 초음파 업로드와 성장 갤러리
- AI 다이어리 생성과 캘린더

### 디바이스 탭

- 공기청정기
- 스탠바이미
- 조명
- 허브와 3D에서 실행된 케어 모드의 시각적 결과

## AI 다이어리

다이어리는 현재 상태, 역할, 임신 주차와 최근 7일의 허브 대화·케어 모드·기기 로그·성장 기록을 종합합니다.

- 임신 준비중: 몸과 마음, 생활 리듬 중심
- 임신중 아내: 본인의 몸과 아기 성장 중심
- 임신중 남편: 배우자를 살피고 함께 준비한 기록

OpenAI 호출이 실패하거나 키가 없으면 상태·역할별 fallback 문구를 사용합니다.

## 시연 범위에서 제외되는 것

- 케어 탭과 메뉴 탭 복구
- 상태 변경 때 새 창 열기
- `localStorage`만으로 서로 다른 기기를 동기화하는 방식
- 의료 진단으로서의 초음파 분석
