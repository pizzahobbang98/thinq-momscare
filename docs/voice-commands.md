# 음성 명령

ThinQ Mom 음성 기능은 자유형 만능 챗봇이 아니라, 현재 시연 범위의 케어 실행과 생활 안내에 집중합니다.

## 진입 방식

| 위치 | 방식 |
| --- | --- |
| 모바일 앱 `/` | 하단 가운데 HUB 버튼을 길게 누르고 말하기 |
| 3D 시뮬레이터 `/simulation-3d/index.html` | "하이 마더" wake word 이후 말하기 |
| AI Hub 패널 `/hub` | 예시 문구 또는 음성 실행 |

## 처리 흐름

```text
음성 입력
  -> OpenAI STT
  -> 현재 임신 상태, 역할, 주차, 후보 텍스트와 함께 의도 분석
  -> 케어 모드 / 기기 제어 / 생활 안내 / 안전 안내 결정
  -> TTS 응답
  -> 공유 상태, 3D, StandbyMe, 공기청정기, 전구 동기화
```

## 지원 범위

- 임신준비중 5개 모드
- 임신중 6개 모드
- 공기청정기 ON/OFF
- 전구 ON/OFF
- 기본 모드 복귀
- 좋은 아침/시간/날짜
- 생활 케어 질문
- 안전/응급성 발화 안내
- 범위 밖 질문에 대한 안내 응답

## 임신준비중 테스트 문구

| 모드 | 테스트 문구 | 조명 대표색 |
| --- | --- | --- |
| 컨디션 밸런스 | 오늘 컨디션이 별로야. | `#FF8A00` |
| 수면 리듬 | 오늘은 푹 자고 싶어. | `#003CFF` |
| 마음 환기 | 집에만 있으니까 너무 답답해. | `#FFCC00` |
| 휴식 준비 | 너무 지친다. | `#FF4E42` |
| 둘의 저녁 | 예쁜 곳에서 저녁 먹고 싶어. | `#C4004B` |

## 임신중 테스트 문구

| 모드 | 테스트 문구 | 조명 대표색 |
| --- | --- | --- |
| 입덧 케어 | 냄새 때문에 너무 힘들어. | `#00B8FF` |
| 수면 케어 | 왜 이렇게 잠이 안들지. | `#5B1FFF` |
| 가사 케어 | 몸이 너무 무거워. | `#A6FF00` |
| 바다 휴양 | 시원한 바다 보고 싶어. | `#00C2A8` |
| 숲 휴양 | 조용한 숲에 가고 싶어. | `#007A2A` |
| 도시 휴양 | 도시 야경 보고 싶어. | `#A100FF` |

## 기기 제어 문구

| 기능 | 예시 |
| --- | --- |
| 공기청정기 켜기 | 공기청정기 켜줘. |
| 공기청정기 끄기 | 공기청정기 꺼줘. |
| 전구 켜기 | 전구 켜줘. |
| 전구 끄기 | 전구 꺼줘. |
| 기본 복귀 | 기본 모드로 바꿔줘. |

## 생활 안내 문구

| 범위 | 예시 |
| --- | --- |
| 아침 안내 | 좋은 아침이야. |
| 시간 | 지금 몇 시야? |
| 날짜 | 오늘 날짜 알려줘. |
| 생활 케어 | 오늘 뭐부터 하면 좋을까? |
| 안전 안내 | 출혈, 심한 통증, 호흡곤란, 심한 어지러움처럼 위험 신호가 포함된 발화 |

안전/응급성 발화는 진단하지 않고 의료진 상담 또는 긴급 도움 요청을 안내합니다.
