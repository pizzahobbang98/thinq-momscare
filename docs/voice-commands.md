# 음성 명령

ThinQ Mom 음성 기능은 임신 상태와 역할에 맞춘 케어 실행형 인터페이스입니다. 자유형 만능 챗봇처럼 모든 질문을 처리하는 것이 아니라, 현재 시연 범위의 케어 실행과 생활 안내에 집중합니다.

## 진입 방식

| 위치 | 방식 |
| --- | --- |
| 메인 앱 `/` | 하단 가운데 HUB 버튼을 누르고 말하기 |
| 3D 시뮬레이터 `/simulation-3d/index.html` | “하이 마더” wake word 이후 말하기 |

## 처리 흐름

```text
음성 입력
  -> OpenAI STT
  -> 현재 상태·역할·임신 주차와 함께 의도 분석
  -> 케어 모드 또는 안내 응답 결정
  -> TTS 음성 반환
  -> 3D 장면, 공기청정기, 전구, 앱 상태 동기화
```

## 지원 범위

- 임신준비중 5개 모드
- 임신중 6개 모드
- 공기청정기 ON/OFF
- 전구 ON/OFF
- 기본 모드 복귀
- 좋은 아침이야
- 시간/날짜
- 생활 케어 질문
- 안전/응급성 발화 안내

## 임신준비중 테스트 문구

| 모드 | 테스트 문구 | 조명 대표색 |
| --- | --- | --- |
| 컨디션 | 아침 컨디션을 맞춰줘. | `#FF8A00` |
| 수면리듬 | 잠을 잘 자게 도와줘. | `#003CFF` |
| 마음환기 | 기분을 바꾸고 싶어. | `#FFCC00` |
| 휴식준비 | 편하게 쉬고 싶어. | `#FF4E42` |
| 둘의저녁 | 우리 둘의 저녁을 준비해줘. | `#C4004B` |

## 임신중 테스트 문구

| 모드 | 테스트 문구 | 조명 대표색 |
| --- | --- | --- |
| 입덧 모드 | 음식 냄새 때문에 속이 안 좋아. | `#00B8FF` |
| 수면 모드 | 잠이 잘 오게 해줘. | `#5B1FFF` |
| 가사 케어 | 빨래와 청소를 도와줘. | `#A6FF00` |
| 바다 모드 | 바다 분위기로 바꿔줘. | `#00C2A8` |
| 숲 모드 | 초록색 나무 보고 싶어. | `#007A2A` |
| 도시 모드 | 도시 야경을 보여줘. | `#A100FF` |

## 기기 제어 문구

| 기능 | 예시 |
| --- | --- |
| 공기청정기 켜기 | 공기청정기 켜줘. |
| 공기청정기 끄기 | 공기청정기 꺼줘. |
| 전구 켜기 | 전구 켜줘. |
| 전구 끄기 | 전구 꺼줘. |
| 기본 복귀 | 기본 모드로 바꿔줘. |

## 생활 안내 문구

| 범위 | 예시 |
| --- | --- |
| 아침 안내 | 좋은 아침이야. |
| 시간 | 지금 몇 시야? |
| 날짜 | 오늘 날짜 알려줘. |
| 생활 케어 | 오늘 뭐부터 하면 좋을까? |
| 안전 안내 | 출혈, 심한 통증, 호흡곤란, 심한 어지러움처럼 위험 신호가 포함된 발화 |

안전/응급성 발화는 진단하지 않고 의료진 상담 또는 긴급 도움 요청을 안내하는 방향으로 응답합니다.

현재 기준: Vercel 시연 기능만 반영.
