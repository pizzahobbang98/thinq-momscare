# 수동제어

수동제어는 모바일 앱 `/`의 하단 탭에서 접근합니다. 음성 없이 같은 케어 실행 흐름을 눌러서 확인하기 위한 시연 화면입니다.

## 빠른 수동 조절

현재 사용자 상태에 따라 버튼 구성이 달라집니다.

### 임신준비중

| 버튼 | 실행 문구 | 결과 |
| --- | --- | --- |
| 컨디션 밸런스 | 오늘 컨디션이 별로야 | 컨디션 밸런스 모드 |
| 수면 리듬 | 오늘은 푹 자고 싶어 | 수면 리듬 모드 |
| 마음 환기 | 집에만 있으니까 너무 답답해 | 마음 환기 모드 |
| 휴식 준비 | 너무 지친다 | 휴식 준비 모드 |
| 둘의 저녁 | 예쁜 곳에서 저녁 먹고 싶어 | 둘의 저녁 모드 |

### 임신중

| 버튼 | 실행 문구 | 결과 |
| --- | --- | --- |
| 입덧 케어 | 냄새 때문에 너무 힘들어 | 입덧 케어 |
| 수면 케어 | 왜 이렇게 잠이 안들지 | 수면 케어 |
| 가사 케어 | 몸이 너무 무거워 | 가사 케어 |
| 바다 휴양 | 시원한 바다 보고 싶어 | 바다 휴양 |
| 숲 휴양 | 조용한 숲에 가고 싶어 | 숲 휴양 |
| 도시 휴양 | 도시 야경 보고 싶어 | 도시 휴양 |

버튼 실행은 3D 장면, StandbyMe 화면, 조명 대표색, 공기청정기 상태, 모바일 기기 카드를 함께 갱신합니다.

## 토글

| 토글 | 실제 기능 |
| --- | --- |
| 공기청정기 | LG ThinQ 공기청정기 전원 ON/OFF |
| 거실 조명 | Philips Hue/Bluetooth 전구 ON/OFF |
| Hue Bluetooth 연결 | 지원 브라우저에서 전구 직접 연결 |

## 기기 카드

| 카드 | 현재 역할 |
| --- | --- |
| 공기청정기 | 실제 ThinQ 상태, 전원, 모드, 미세먼지 표시 |
| 거실 조명 | 실제/중계 전구 전원과 대표색 표시 |
| 스탠바이미 | 현재 모드에 맞는 화면 콘텐츠 표시 |
| 냉장고 | 입덧/식사 케어 상황 표현 |
| 로봇청소기 | 가사 케어 상황 표현 |
| 워시타워 | 빨래/가사 케어 상황 표현 |

## 기본 화면 복귀

3D 시뮬레이터에서 기본 모드로 복귀하거나 idle 복귀가 발생하면 다음 상태가 초기화됩니다.

- 현재 케어 모드
- 3D routine
- 최근 음성 명령 표시
- 수동제어 카드의 활성 모드
- StandbyMe 모드 화면
- 거실 조명 기본 대기색
