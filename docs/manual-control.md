# 수동제어

수동제어는 메인 앱 `/`의 하단 탭에서 접근합니다. 현재 시연에서는 실제로 사용하는 제어와 카드만 문서화합니다.

## 빠른 수동 조절

상태에 따라 버튼 구성이 달라집니다.

### 임신준비중

- 컨디션
- 수면리듬
- 마음환기
- 휴식준비
- 둘의저녁

### 임신중

- 입덧 모드
- 수면 모드
- 가사 케어
- 바다 모드
- 숲 모드
- 도시 모드

빠른 수동 조절 버튼은 음성 명령과 같은 의도 분석/실행 흐름을 사용해 3D 장면, 공기청정기, 전구, 공유 상태를 함께 바꿉니다.

## 토글

| 토글 | 기능 |
| --- | --- |
| 공기청정기 ON/OFF | 실제 LG ThinQ 공기청정기 전원 토글, 카드 상태 동기화 |
| 전구 ON/OFF | 실제 Philips Hue/Bluetooth 전구 전원 토글, 거실조명 카드 상태 동기화 |

## 기기 카드

| 카드 | 현재 역할 |
| --- | --- |
| 거실조명 카드 | 전구 ON/OFF, 밝기, 대표색 표시 |
| 스탠바이미 카드 | 모드별 화면 콘텐츠 표시 |
| 냉장고 카드 | 입덧/냄새 케어 상황 표현 |
| 로봇청소기 카드 | 가사 케어 상황 표현 |
| 워시타워 카드 | 가사 케어 상황 표현 |
| 공기청정기 카드 | 실제 ThinQ 상태와 케어 모드 표시 |

## 기본 화면 복귀 동기화

3D 시뮬레이터에서 기본 모드로 복귀하거나 10초 idle 복귀가 발생하면 다음 상태가 함께 초기화됩니다.

- 현재 케어 모드
- 3D routine
- 최근 음성 명령 표시
- 수동제어 카드의 활성 모드
- 거실조명 기본 대기 상태
